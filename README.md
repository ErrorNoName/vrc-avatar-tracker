<div align="center">
  
  **The Ultimate Hub for Your VRChat Avatars**
  <br />
  Securely login, track, search, and manage your favorite VRChat avatars locally with a sleek, modern, and lightning-fast database.
</div>

## 🌟 Features

- 🛡️ **Secure 2FA Login**: Connect seamlessly to your VRChat account. We fully support Two-Factor Authentication (OTP/Email). Your session data remains safely stored on your local machine.
- 🔍 **Powerful API Search**: Browse through millions of public avatars on VRChat. Instantly search by name or author directly using the official VRChat API.
- 💾 **Local Database**: Save avatars to your custom favorites lists. Your collections are stored offline in a highly optimized database, ensuring you never lose a cool avatar again.
- 🎨 **Modern Interface**: Designed with a premium, sleek dark/light theme, built using React and Tailwind CSS.
- ⚡ **Portable**: Download and run immediately. No complex installation required.

## 🚀 Quick Start

### Download

## 🛠️ Technology Stack

- **Electron**: Desktop application framework.
- **React & TypeScript**: Front-end UI and logic.
- **Zustand**: Fast and scalable state management.
- **Tailwind CSS**: Modern utility-first styling.
- **Axios**: HTTP requests to the official VRChat API.
